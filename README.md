# Movie-Review-Platform


## 프로젝트 개요


### 프로젝트 명


영화 리뷰 플렛폼


### 프로젝트 기간 


24.11.15~24.12.10


### 프로젝트 목적 


영화에 대한 정보를 제공 하고, 사용자가 개인적인 리뷰를 작성하여 공유 할 수 있는 온라인 플랫폼.



### 프로젝트 필요성 


영화에 대한 다양한 정보를 한 곳에서 제공하고, 다른 사람들과 소통할 수 있는 공간을 제공 함으로써 영화 관람 경험을 향상 시킴
